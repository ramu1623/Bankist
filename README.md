# Bankist App 💰

This is a front-end banking simulation project built with HTML, CSS, and JavaScript.

## 🔗 Live Demo
👉 [Click here to view the project](https://ramu1623.github.io/Bankist/)

## ✨ Features
- Login simulation
- Transaction history
- Account balance calculation
- Auto logout timer

## 📁 Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)
